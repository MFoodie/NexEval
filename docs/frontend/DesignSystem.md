# NexEval 设计系统

## 1. 设计原则
- 现代、智能、简约：界面要克制，信息层级清晰。
- 数据优先：图表与指标高可读，装饰性元素弱化。
- 有呼吸感：留白充足、阴影柔和、渐变轻量。
- 企业级可信感：稳重色彩与一致的组件规范。

## 2. 色彩体系（Color Tokens）
### 品牌色
- 主色：#2A5CFF（智能蓝）
- 主色浅阶：#E9EFFF
- 点缀色：#00C2FF（科技青）

### 中性色
- 背景：#F6F8FC
- 表面：#FFFFFF
- 分割线：#E6ECF5
- 正文：#1F2A3D
- 次级文字：#7B8AA6
- 禁用文字：#B7C2D3

### 语义色
- 成功：#22C55E
- 警告：#F59E0B
- 错误：#EF4444
- 信息：#3B82F6

## 3. 渐变规范
- 主渐变（按钮/头部）：linear-gradient(135deg, #2A5CFF 0%, #00C2FF 100%)
- 卡片顶边：linear-gradient(90deg, rgba(42,92,255,0.18), rgba(0,194,255,0.08))

## 4. 字体与排版
- 主字体："Plus Jakarta Sans"，"Inter"，system-ui
- 数字与图表："IBM Plex Mono"，monospace
- 字号体系：
  - Display：28/36
  - H1：24/32
  - H2：20/28
  - Body：14/22
  - Caption：12/18
- 字重：
  - 标题：700
  - 模块标题：600
  - 正文：400

## 5. 间距体系
- 基础单位：4px
- 常用间距：8/12/16/20/24/32/40
- 卡片内边距：20px（桌面），16px（移动）

## 6. 圆角与阴影
- 圆角：12px（大卡片）、10px（常规）、8px（表单控件）
- 阴影：
  - 卡片：0 6px 24px rgba(16, 24, 39, 0.08)
  - Hover：0 10px 30px rgba(16, 24, 39, 0.12)
  - Elevation：0 14px 40px rgba(16, 24, 39, 0.18)

## 7. 图标规范
- 风格：线性、圆角端点、1.5px 描边
- 推荐：Lucide 或 Remix Icon
- 尺寸：16/20/24

## 8. 动效规范
- 缓动：cubic-bezier(0.2, 0.8, 0.2, 1)
- 时长：120ms（微交互）、200ms（卡片 Hover）、320ms（页面进入）
- 进入动画：淡入 + 轻微上移（translateY(6px)）

## 9. 核心页面布局（评估看板）
- 顶部：全局导航 + 3~4 个 KPI 快速指标
- 主体：
  - 左侧筛选区占 20%（固定宽度）
  - 右侧内容区占 80%（趋势图 + 评估列表）
- 右侧结构：
  - 上部：趋势图（折线 + 预测点）
  - 下部：评估列表（表格 + 行内指标卡）
- 交互：筛选变更触发右侧刷新，配合骨架屏与渐隐过渡
